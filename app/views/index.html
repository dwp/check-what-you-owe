05 - Combined start page

{{ govukWarningText({ text: "Prototype archived and not tested ", iconFallbackText: "Warning" }) }}

One link on the letter points towards a single start page, which filters out users based on the task they are trying to do.

This wasn't taken any further, because our user research shows personas who have an affordable debt to pay off, don't want a continuing relationship with the DWP, they just want to pay off their debt. If we did this route, then all users would be required to use a login rather than a one time access code, which could cause drop off for this type of user.
